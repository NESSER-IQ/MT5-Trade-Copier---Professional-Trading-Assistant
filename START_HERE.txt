تم تطوير نظام متكامل لنسخ إشارات التداول من التليجرام إلى MetaTrader 5 تلقائياً
مع ميزات متقدمة مثل Trailing Stop ودعم 10 مستويات أخذ ربح.

================================================================================
                            🚀 البدء السريع
================================================================================

الخطوة 1: تثبيت المكتبات
---------------------------
افتح Command Prompt في مجلد المشروع واكتب:

    pip install -r requirements.txt

أو:
    pip install telethon MetaTrader5 customtkinter matplotlib pandas numpy python-dotenv


الخطوة 2: إعداد ملف .env
--------------------------
1. انسخ ملف ".env.example" وأعد تسميته إلى ".env"
2. افتح ملف ".env" وأدخل بياناتك:

   API_ID=12345678
   API_HASH=abcdef1234567890
   PHONE_NUMBER=+966501234567
   MT5_LOGIN=12345678
   MT5_PASSWORD=YourPassword
   MT5_SERVER=MetaQuotes-Demo

🔗 للحصول على Telegram API:
   اذهب إلى: https://my.telegram.org
   سجل دخول → API Development Tools → Create Application


الخطوة 3: تشغيل البرنامج
--------------------------
انقر مرتين على: run.bat

أو اكتب في Command Prompt:
    python main_gui.py


الخطوة 4: الاتصال
------------------
1. في تبويب "الإعدادات":
   - أدخل بيانات التليجرام
   - اضغط "الاتصال بالتليجرام"
   - أدخل كود التحقق المرسل لهاتفك

2. أدخل بيانات MT5:
   - رقم الحساب
   - كلمة المرور
   - اسم الخادم
   - اضغط "الاتصال بـ MT5"


الخطوة 5: إضافة قنوات
-----------------------
1. اذهب إلى تبويب "القنوات"
2. أدخل رابط القناة أو @username
   مثال: https://t.me/tradingchannel
   مثال: @forex_signals
3. اضغط "إضافة"


الخطوة 6: البدء!
-----------------
✅ سيبدأ البرنامج باستقبال الإشارات تلقائياً
✅ ستنفذ الصفقات على MT5 تلقائياً
✅ نظام Trailing Stop سيعمل تلقائياً
✅ تابع صفقاتك من تبويب "الصفقات المفتوحة"

================================================================================
                        📈 كيف يعمل Trailing Stop؟
================================================================================

مثال على صفقة شراء XAUUSD:
- الدخول: 3330
- TP1: 3335 | TP2: 3340 | TP3: 3345
- SL الأصلي: 3320

التنفيذ:
1. السعر وصل 3335 (TP1) → SL ينتقل إلى 3330 (نقطة الدخول)
   ✓ الصفقة الآن محمية من الخسارة!

2. السعر وصل 3340 (TP2) → SL ينتقل إلى 3335 (TP1)
   ✓ ضمنا ربح 5 نقاط على الأقل!

3. السعر وصل 3345 (TP3) → تُغلق الصفقة
   ✓ ربح كامل!

إذا عاد السعر في أي وقت، ستُغلق الصفقة عند SL الجديد
وليس SL الأصلي، مما يحمي أرباحك! 💰
